# 📊 Student Performance EDA


## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a student performance dataset to uncover patterns, relationships, and key factors affecting academic scores. The goal is to derive meaningful insights that can help improve student outcomes.


## 🎯 Objectives

* Analyze student performance data
* Identify factors affecting academic scores
* Perform data cleaning and preprocessing
* Visualize trends and correlations
* Generate actionable insights


## 📊 Dataset Information

* Source: Public dataset (Kaggle)
* The dataset contains information about students such as:

  * Gender
  * Parental level of education
  * Lunch type
  * Test preparation course
  * Math, Reading, and Writing scores


## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab


## 🔍 Analysis Performed

* Data Cleaning
* Handling Missing Values
* Univariate Analysis
* Bivariate Analysis
* Correlation Analysis
* Data Visualization


## 📈 Key Insights

* Students who completed the test preparation course scored higher
* Parental education level positively impacts student performance
* Female students performed better in reading and writing
* Students with standard lunch performed better than others
* Strong correlation observed between reading and writing scores
